Source Code 使用方式

说明：这里主要是开发相关的配套文件，如字段设计文件，表单设计器的源文件（使用方式为 下载json–导入json–预览页面）；后续根据精力会添加new_table.sql :)

  1. 右键另存为当前 json文件，如 造价审查申请 高仿OAjson，或者导入时复制粘贴（建议使用前者）
  2. 进入页面：http://cdn.kcz66.com/k-form-design.html
  3. 选择 导入json 文件
  4. 加载json文件
  5. 点击预览按钮，可根据需要调整预览窗口宽度，我自用的为1200
