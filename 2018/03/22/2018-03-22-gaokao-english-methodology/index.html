Gaokao English Methodology

发表于 2018-03-22 更新于 2019-09-16 分类于 English

现在的工作是帮助已经过了儿童阶段的高三学生（以文科生、艺术生为主），从
比较薄弱的基础环节下来准备高考英语。鉴于语言学习的特殊性和教育行业的特殊性，
从教学成果比较好的老师那里取经是非常有必要的。

作者：Lluvia
链接：https://www.zhihu.com/question/41882061/answer/92732473
来源：知乎

英语老师一枚，多次把80几分学生带到130，来答本题。

首先题主能不能先把自己平时考试和一模的成绩报一下。学渣这个定义太宽泛了～～～我为什么这么强调分数呢？题主也说了，自己只有两个月，定位是首先要做的，最害怕东一榔头西一棒槌的学生了。【还有私信过我的孩子们，如果想让我分析一下成绩偏低的原因，最好描述一下自己的年级、成绩，哪部分考的不好，做过什么努力。只说成绩差问我怎么办，很难回答吖～】

说回正题哈～先记住这点，对自己的分数段要十分了解，考70分时就做好70分提到100分该做的事情，不要去做130分的学生干的事。等考到100分时再做能提高到130分该做的事儿。好啦～下面是2个月英语复习咱们该做的事儿。

一、0～80 分如果你是80分以下的学生，你的目标是提分到80～120分段。给你个复习策略，请严格按照（一），（二），（三）的顺序。（一）学习宏观语法。图片奉上～请对照图片，花半个小时搞明白语法是什么

（二）背高频词688个（网上可以搜到，搜不到私信我）用3天～10天不等的时间，背下来（其中简单的包括person、difficult等，难的如suffer、review等）。高频词背到什么程度呢？看见英文大概知道它中文是什么意思就可以了，不需要会拼写。

（三）用阅读背单词（这个阶段不需要做题，只看文章即可）拿出五三（或者任何一本有高考阅读原题的练习册），挑出其中的记叙文（记住只要记叙文！！！）25篇。把这25篇记叙文完全读懂。阅读方法…是…

（1）把不会的单词标记出来，查一下意思，写在本子上；

（2）阅读文章内容，划出不会翻译的句子；

（3）分析你不会的句子，可以攒够了直接去问老师，或者学霸，这个语法结构是怎么样子的；

（4）把所有抄上本子的单词和你之前688高频词一起复习一下。这个阶段搞完了之后你可以用全国卷难度自测一下，肯定突破100分了，那你就进入下一个阶段了。

二、80～120分 的学生，请按照以下方法（严格按照顺序）学习。

（一）刷改错题+语法填空能够到100分左右的学生都是属于有语法概念，但是不能和考点结合的毛病。说实话，高考语法学的多，考的少。整张卷子，就改错和语填两个考语法的题，而且考的都是初中语法，考点重复率极高。尤其改错，简直是用第六感在做题（我把题刷多了，看见就会的感觉叫做第六感），你每天刷两篇改错，10天之后就能保证准确率80%以上。看留言，好像很多人都不知道改错怎么刷题吖，贴上我的方法，有兴趣可以看下～高考的英语短文改错有什么方法提高？ - Lluvia 的回答

（二）做阅读这个阶段的阅读就不是为了提高词汇量了，而是为了做题准确而读。想办法把题做对，你就胜利了。但是，仍然要选择高考阅读原题，而且记叙、议论、说明文都要做，文章内容可以不完全理解，但是题要做对。为什么一定要选阅读呢？因为，阅读的分值是卷面最高，是高考英语的大半壁江山，我们经历了第一阶段已经有了一定的词汇量，这个时候练阅读提分会非常明显（表问我为什么哦，拿学生做实验试出来的）坚持30篇阅读，你就120～130分了，实践过无数次，都是有效的。很多人问我阅读怎么提高准确率，我们首先需要把题目分成【细节题、推断题、主旨/标题题】，然后再去原文找答案，准确率会高很多，具体就不在这里答了，下周会写一下阅读技巧，有兴趣可以关注一下。

三、120～135分，请参考以下办法。刷完形填空 + 练字 + 写作文（最好找老师面批作文，学霸也帮不了你的）完形填空我写过一个详细的方法，有兴趣可以去看看。完形填空有没有答题技巧？ - Lluvia 的回答今天回答了作文提分方法，有兴趣的孩子可以看看～如何写出高分的高考英语作文？ - Lluvia 的回答至于135分以上的孩子，我就不管了。【补】有人问我为什么要强调练字，而且只对高分数段的孩子做要求～对于成绩没那么好的孩子，练字要浪费时间了，不如其他题提分快，可以先做题为主。但是对于想考到135分的学生，作文要拿到20分以上，说实话，其他部分很难再提高了，同样的作文，工整的字和烂字就是23～18分的差距，贴两张我自己写的字，给大家做对比参照
