如何看待希拉里「中国不会超越美国，因中国明星和中国富豪喜欢拿钱移民」的预测？

发表于 2020-03-05 更新于 2020-03-04 分类于 Report

她输了几次，和她说话对不对是两回事。

先看看她这个预测的数据来源是啥：

目前，能找到的，方便公开的数据是招商银行和贝恩资本共同发布的《2011中国私人财富报告》。

九年过去了，情况是更恶化了还是变好了？

http://www.safe.gov.cn/

看看历年外汇储备和国际收支的的情况，大致就有数了。相关的知识见本科《金融学》教材，不难，理解经常账户和投资与金融账户的概念就知道了。这些知识大概率不能帮你直接赚钱，但能帮你清理政治正确迷雾，看到世界相对真实的一小部分。

至于“一个国家的兴衰，完全取决于上层1%，而不是金字塔的基座……”究竟对还是不对？很简单，看组成金字塔的基座的劳动者，是否有真正的共同政治纲领，是否有真正的强有力组织，只有做到这两点，才有谈集体命运的资格，否则人数再多，也只是乌合之众和韭菜肥料集群，不可能改变集体的命运。这是很简单的政治常识。

那么，中国的这个群体是否符合这个条件？你说呢?连996问题都解决不了的群体，有那个能量吗？

如果以这次疫情为时代背景拍部电影，怎么拍，才有可能拿到明年的奥斯卡呢？

作者：邓铂鋆
链接：https://www.zhihu.com/question/372460276/answer/1050916792
来源：知乎
著作权归作者所有。商业转载请联系作者获得授权，非商业转载请注明出处。

美国流感季来势凶猛，加州的一位伊朗移民女医生发现大批来自社区的不明原因肺炎病人。病人的甲流乙流咽拭子检测均为阴性，同时相当数量的病人来自东方某国。

女医生联系CDC，怀疑网络流传的东方奇疫传到了美国。老白男官僚傲慢的表示这就是流感，女医生费尽口舌终于从CDC得到了一批东方奇疫检测试剂盒，检测发现病人均为阴性。

女医生联系到了花生屯的一位高官，她已经分手的同性情人，发现自己的试剂盒全是假货。老情人帮女医生拿到了走私的试剂盒检测，发现病人都是确诊病人。老情人把情况报告给老白男总统，要求总统采取包括实施紧急状态和禁止东方国家人员入境在内的一系列措施。

老白男总统更刚愎自用，吼了一通他会隔离一切，没有人比他更懂病毒。然后一切照常，总统忙于竞选连任，他寄期望跟东方大国的经济合作，要在白宫接待求见的东方大国经贸考察团。

白宫晚宴上，东方来的客人发病晕倒。一位剧烈咳嗽的东方富商说出实情，他们是来美国保命的。老情人及时掏出N95口罩，保护总统离开。疫情愈演愈烈，CDC终于捂不住盖子了。老情人从亚裔理工男前男友那里得知，假如疫情不加控制，一个月后全国会有超过两千万人感染。一些地方开始发生骚乱，老白男总统震怒。

老白男CDC负责人失势，新上位的黑人大叔主管透露了一个消息：半年前，一位美国医生前往东方大国，她在那里治愈了大量病人。如果美国获得了康复病人的血清，血清可以帮助美国迅速制出疫苗。这位美国女医生就是亚裔理工男的亚裔前妻。

在老情人的帮助下，理工男、女医生跟随一小队军人秘密潜入东方大国。军人不PC，歧视理工男和女医生，很快就领便当收工。东方大国此时已经是一片末日景象。理工男和女医生来到了女医生的营地，发现这里就是世外桃源。当地人迷信，不愿意献出血液。在亚裔女医生的感召下，几乎全部的康复病人都献出了血液。

理工男和亚裔女医生在营地中磕磕绊绊的提取了血清。然而，早已守候多时的东方大国军队前来抢夺。为首的赵将军毫不掩盖要用疫苗控制世界的野心。理工男领便当收工，亚裔女医生本来舍不得病人，但是不得不离开，留下一句“我还会回来的”。两位女医生在当地群众和那位咳嗽富商的帮助下成功登上夯炕近海的美国军舰，两人相拥热吻。

花生屯，老白男总统的丑闻爆发。借助疫苗和女医生在东方大国取得的经验，老情人和CDC黑人主管力挽狂澜，在解决美国疫情的同时向全世界伸出援手。一位金发的老年女性政客提出要支持两人竞选总统。两位女医生回到东方大国，在美国军队的护卫下运营一个医疗营地，亚裔女医生跟当地群众笑着说“我说过我会回来的”。虽然那里的街头仍然萧条，但是人们都摘下了口罩，露出健康的笑容。
