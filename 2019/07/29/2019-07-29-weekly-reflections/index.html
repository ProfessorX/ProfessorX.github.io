Weekly Reflections 20190729

发表于 2019-07-29 更新于 2021-09-10 分类于 Report

苟日新，日日新，又日新。

减肥大业一刻也不能耽误，更关键的是要均衡营养合理膳食。严厉杜绝暴饮暴食情况再次发生。

Pros

  • 本周业务范畴内的东西应该比较平稳有效的完成了。
  • 本周的健身天数为6天，算是历史比较好的成绩。

Cons

  • 本周提出严厉批评的是：周六在表哥家里暴饮暴食，导致大半个月的减肥成果白费，体重上升4斤左右。惩罚是坚持每周5-6天的有氧运动不动摇。
  • 本周末的时间没有有效的利用起来。周六荒废在游戏和电视剧上面，周日起床时间比较晚，然后下午午休的时间太长。
  • 本周的工作效率一般，还有在开会的时候睡觉的情况，很危险。

TODO

  • 继续熟悉国内和海外的服务业务实操。
  • 磨练技术，看完java技术类书籍，开始编程。
  • 学理不学文，一辈子混底层。定好大致的方向，开始利用健身的时间在Coursera平台跟课。
  • 做好考国内CPA证书的准备。
  • 三件大事，健身，技术（Introduction to Java Programming)，公司财务（Master of Science in Accounting)。
