Dota Underlords Research

发表于 2019-07-16 更新于 2021-09-10 分类于 Report

here are some Chinese guide docs for the new game of Steam(Valve) Dota Underlords.

终于上尖塔领主啦，写一下我上分的心得 (其实跟别人的攻略差不多)

接下来是上分心得(仅适用于本周，因为马上要有大更新了，所以我觉得大家欲上分从速，因为这个版本虽然玩着无聊，但要想上分还是挺简单的。)

简而言之还是那句听到吐的话：转大招。
第二重要的就是保持前期连胜。(感觉像废话)
分享一些保连胜的小技巧：
分为两部分：(1)重要的前期棋 (2)重要的前期被动。

重要的前期棋子：排名不太分先后
(小鹿，大树)，两个缺一个都不太厉害，所以放在一起。两个人的恢复能力经常能帮你好似对面。
蝙蝠，可以配合巨魔被动或再上一个骑士减伤。两星的蝙蝠油叠起来伤害高的吓人，甚至可以配合跳刀或者放到第一排(前几回合)。
小小，对我来说，小小是2星单体最强一费棋子。有的时候可以用到30多回合。
敌法，攻击力虽然不高，但有魔抗，能阻止对方前排棋子(大树，小小等)放出技能，还是很关键的。
小黑，无情的重要组成部分。大家都知道的，前期配合屠夫或者死骑都不错。

重要的前期被动：排名不太分先后
虹吸魂管，虽然前期用处不大，但看到一定要选。因为中期有用，后期更有用。
不朽之守护，打到后期，经常出现8个人都在，但第一名也就30血的情况。又一个肉山盾，经常可以排名提前2-3名。而且这个物品也让你心里更有底，可以放心卖血升级。(后面会提到)
失去天恩，我觉得这个被动最厉害的是让阵容更加灵活。有了这个，就不一定非要盯着那几个无情棋子买了，所有人类都可以用。减少脸黑几率。想上分必选。
奥数鞋，后期让大招比别人先放出来的重要道具。
联手突袭，有一个这个，上两个巨魔，两个巨魔中间夹一个主要dps。前期连胜稳稳的。
闪烁匕首，这版本后期最强道具。前期可以给一些技能厉害的英雄(狼人，船长，小小，又一次我给了CK，效果也不错，总之前期给谁很灵活)。后期给萨尔，给炸弹，都是很好用的。
尖牙利爪，由于小鹿和狼人很强，如果选得到这两个棋子的话，这个被动也很厉害。可以占总输出的20%左右。
走私犯和有钱人的烦恼，这两个都是在没什么可选的时候选一选，也不错。尤其是第一回合结束后，有个有钱人的烦恼可以让你有更大几率在后面选到有用的被动。

中期心得：
中期重要的被动只有一个，那就是：高级别罪犯。
剩下的就是运营：
在我打的局里面，不管是顺还是不顺。我都会4回合升5(除非第四回合刚好刷出关键棋子需要钱买)，9回合升6，17回合之前升7(取决于顺不顺)，21回合升8，25回合之前升9，29回合升10.
又一个小技巧，我会观察别人的升级情况，尽量保证自己是等级最领先的哪个。如果有几人跟你一起到9，也不要慌。看他们的钱和他的用钱习惯。如果只有一两个人丧心病狂的搜棋子，那你就安心上10，如果大家都丧心病狂，那没办法了，必须得开始搜了。因为每个5块钱棋子只有10个，被人搜完就没了。
我之前出现过，刷新只显示4个棋子，不知道是不是某个棋子被搜完的意思。

因为这版本大招实在是太厉害了，升级收益太大。你前期那些棋子，到后面总归都要卖掉换成5块钱的棋子的。所以升级他们没什么必要。有的时候我因为升级，场上战斗力不强，被人入到只剩20多血，但我钱多，等级高(如果有肉山盾，就更不慌了)。到10之后，搜一波，上几个5费棋子，之后就再也没有人能打过我了。有几次10几血，跟7，80血。一直把那个人打到死。

后期阵容推荐：
如果有失去天恩的话：
跳刀炸弹就是绝对主力。其他的话，推荐4术士。谜团，炼金，萨尔和死灵法师。然后比较重要的就是巫妖和潮汐(对面法师多的话，可以上美杜莎和潮汐加些魔抗)。这样一共是7/8个棋子。
剩下2/3个棋子就上人类和无情。4无情就够了，不用非要凑6无情。
人类推荐冰女，让你的大招放的更快。

如果没有失去天恩的话：
没有很多无情羁绊的话炸弹人就没那么强。可以考虑用飞机当主力，4术士，然后走法师流，冰女巫妖再加一个法师。如果你骑士多的话，推荐这里加精灵龙和龙骑。不过说实话，加什么区别都不大，dps榜上基本就只会有巫妖，飞机和谜团。

总而言之一句话，所有的都为大招做嫁衣。不管是前期的尽力连胜，还是中期的升级，还是后期的羁绊选择。

后期的装备推荐：
黑皇：必选。给萨尔或者没有跳刀的炸弹，或者潮汐。保证控制/伤害能打出来。
刷新：有两星潮汐或者船长可以给。
笛子：对面有法师的时候增加存活率。
辉耀：伤害很高，但到大后期，战斗时间超短，也没什么大用。
羊刀：站位针对，控制对面关键棋子。
+1人口，强袭，血棘：因为是5级装备，不好拿，放在一起说了。血棘针对对面大招，强袭抵御对面无情，+1人口经常可以增加一个羁绊或者再上一个5块钱棋子。
邪恶意念：如果场上你认为最强的那个选手有术士/回血的话，选这个有助于你在跟他1v1的时候占上风。(前提是要有无情/失去天恩)

目前想到的就这些，第一次写心得，可能不是很有条理。总之感谢看到这里，并祝大家上分顺利，把把天湖

浅谈这版本冲分首选

上周六称霸五起起伏伏几次，最后一波崩到称霸一，拿了一个第8直接掉两段，今天有时间终于一波打了8把6鸡2第2上了领主

核心思路和加亮的帖子一样，战猎组合，后期经济好就转大招

这版本非常难翻盘，主要有几个原因，一是连败奖励nerf，二是橙卡太强，大部分单卡弱，三跳刀buff，非常适合炸弹，潮汐

所以前期强的有连胜有血量，可以安心攀科技，先到9，10级的刷橙卡，稳住优势，而前期劣势的血量堪忧，经济也不好，为了苟排名不得不在7，8级把钱刷掉，基本失去吃鸡可能。我印象中绝大部分吃鸡的都是前期就优势的，几乎没有逆风翻盘的，这一点我觉得是这版本最大的缺陷

另外炸弹伤害要削弱，所有组合现在上个2级炸弹都很imba，有跳刀就是对面的噩梦

打到高分局几乎就没见过恶魔流，地精流，刺客流，巨魔流，骑士流吃鸡，这几个流派的缺点非常明显，就是因为转大招流麻烦，非常依赖自己组合的特性，很难拆开组
而战猎是最好转大招流的，猎人只需要三猎人就可以，其中两个还可以是潮汐大娜迦，法师流也可以，但是法师流的缺点是前中期可能稳不住

前期的运营方面
3-6级优先战士buff，6-8人口我看卡牌的质量优先6战士buff，3猎人buff其实提升不大，不用强行3猎人上了低质量的卡
打个比方，7级的时候可以6战士+小黑凑个亡灵buff，或者对面法系多，你有大那家+大鱼凑个魔抗。当然也不用硬凑6战士buff，尽量上质量高的卡，这个都可以看局势看自己情况选择

战士的中前期选择方面，屠夫是核心，20回合前有2星屠夫你至少血量安全，其他战士不管谁2星都可以先上，主要是能扛就行。2星屠夫可以说就是前期的key牌，2500血能装一个锁子甲或者先锋盾的话可以扛太久了

猎人的话中前期火枪非常强大，不仅攻击高，而且大招稳定消灭一个，前期搓条2秒也不亏。2星兽王可用，但前10回合都没2星就可以放弃了，大那家一星的话和火枪风行差不多，但有冰眼的就很强了，输出一般但是这版本冰眼减速非常强，可以减速3个，pom我几乎不用，性价比低
中后期，一般是娜迦+小黑+火枪/风行+潮汐，如果位置不够可以下掉火枪/风行，2星的时候大娜迦无敌，版本之子，遗物大炮银月都行，最后血量低的生死局都可以拿圣剑，我用圣剑翻盘过。冰眼丢给火枪或者飞机吧，超远距离减速非常恶心，梅肯笛子这类小黑拿着，小黑输出低就一个挂件辅助

另外中前期不应局限战猎，只要战斗力强都可以用，sf配合ss的双ss流前期战斗力很强，如果你有2级赏金或者2级敌法，在15回合之前战斗力都很不错
前期猎人并不强，所以优先3战士之后，你需要的只是找输出，任何输出高的卡都可以先过度用

对各天赋的感觉
这版本最强的天赋，t0高级犯罪+天恩
大招流无敌的版本，高级犯罪稳居第一
我冲领主的最后一把，前中期血崩，但是20回合拿了一个高级犯罪，我7级的时候白嫖了炸弹，直接跳刀炸弹稳住局面，一波连胜最后吃鸡
失去天恩的强大主要是6亡灵减20甲，我没洗细算过，但是我用的效果感觉伤害几乎翻倍，拿到失去天恩6亡灵猎人流一样可以和大招流一战

另外还有几个职业天赋我觉得是t1级别，
一巨魔buff，拿了这个buff组一个巨魔流中期无敌，不需要一定组巨魔骑，像其他巨魔ss，巨魔战，巨魔战猎中期都很强，而且这版本比巨魔骑还强。至少保你进决赛圈，但吃鸡还要看情况
二悍将，前期有2级兽王就是t0buff，没有的话反而是累赘，剑圣斧头前期能用，后期也是弟弟，双悍将我以前有过，但是最近更新排名的版本后，我从来没拿过，也没见别人拿过，是不是有暗改我不知道
三邪恶意念，完美克制奶d流，ss流
四法师buff很强，即使你只有一个2级lich是法师也性价比贼高
野兽buff我没试过，但是遇到过别人用，也很强

[霸业达成] 白色尖塔领主达成！说说我的路子

我的上分套路就是一种：一切转战猎法再转大招。9人口算是成型，理想阵容是 船长，狼人，屠夫/末日，美杜莎，小黑，潮汐，冰女，光法，巫妖。能拿到失去天恩这就是6亡灵组合，对面的前排就会像纸一样，拿不到失去天恩这就是4人类组合，44%的沉默概率让你很容易抢到先手。

在达成理想阵容的过程中，我们可以有很多的打工仔，什么两星用什么。比如前期的小鹿大树，小小，海民，斧王，剑圣，甚至各种刺客都是可以的。总结起来就是要保证场上的组合是你的最高战斗力，能拿连胜就不要拿连败。
开局之前先明确思路，最终我们的阵容是要朝着大招流走的，所以前期拿的将都是打工仔，不要为了纠结几个对子而搜将，这是在透支中后期的经济。

1-10波，有钱人的烦恼，走私犯，失去天恩，无人能挡，这四个属于必拿天赋，没有的话就拿装备。兽人不在我们的阵容中，所以战斗中铸就不拿。
拿将基本上先拿战士，小鹿大树有就最好，没有也不强求。对于小小和小黑的抉择我是先拿小小，前期的两星小小比两星小黑厉害太多了。骑士来的多也可以考虑用一下，地精千万不要拿，实在是太蠢了喷。

11-15波，这个时候有6人口，应该能凑到双德鲁伊/三猎/三战，这个阶段是经济快速积累阶段，一般不要搜将，为了吃一块钱利息可以考虑卖掉打工仔对子，只留下7人口需要上的将。光法和冰女看情况留，毕竟是要8-9人口才能上的。15波的野怪是有可能出高级别罪犯的，要是能拿到这局基本就稳了。

16-20波，经济好的16波就能上7，经济差的17波也一定要上7，少吃一块钱利息也没事，这个时候少掉点血才是最关键的。然后根据场上战斗力考虑是省人口还是搜一下。如果场上全是一星，那就要搜一下增强战斗力了，不然可能一波就掉个十几滴血。这个时候就留30块存款，其他全花来搜将。如果场上两星多，就提升人口增加战斗力。

21-25波，21波升8，如果就剩30-40血，且战斗力不是全场最高，那推荐把钱花光，法师全卖，就打战猎或者6猎苟名次了。如果血量还健康，战斗力马马虎虎，再决定是搜将还是升人口。这里有个很难抉择的问题：8人口搜不搜美杜莎。8人口搜紫卡概率低，但是升人口又可能会被入，每次我都纠结半天。我的选择是小搜一下，能搜到最好，搜不到就攒钱升9了，通过增加单位来增强战斗力。

26-30波，这个时候阵容基本就成型了，打工仔也都光荣下岗，有法师就上大鱼和美杜莎组双娜迦，有刺客就把一个战士拉后。需要注意的是光法一定要两星才能上场，一星光法甚至没有一个两星海民有用。

31波之后就是决赛圈了，这个时候的理想应该是潮汐先手踩大，船长接上船，美杜莎开大石化，然后光法一个波加巫妖连环霜冻，然后世界就清净了。

要转大招流的话，只有潮汐，美杜莎，小黑，船长，冰女，光法是必需的，其他都可以替换成橙卡。

几张战略卡：萨尔，抢先手的时候非常有用，特别是对面缩成一团，一个两星萨尔能保证你能先放出技能

TA：有一把决赛圈我们两家大招流对轰，对面三星冰女每次都能先放技能，我这边后排勉强能放出技能然后被融化，他那边还剩光法飞机之类的。我直接把小黑什么的卖了，上了两个两星TA，最后场上站着的就剩我这两个TA。最后也是成功吃鸡

炸弹人：可能有人还不知道炸弹人的AI改了，炸弹人现在会在身前放桶而不是身后，而且桶的范围有4格，也就是说放在第一格就能炸到对面后排了，面对抱团阵容时一个炸弹人能让对面上天。

跳刀：跳刀的作用是跳到离你最远的敌方棋子处，当对面有跳刀潮汐时可以把整体阵容跟潮汐对边，然后随便拉一个棋子(比如冰女)放到潮汐的对角线，这样潮汐就会稳定跳到冰女旁边了。

羊刀：对面美杜莎靠前的话把羊刀给到前排，可以秒羊美杜莎，或者跟对面的潮汐/船长对位，羊掉对面控制就好打很多。

不朽之守护：现在效果改成帮你抵挡一次致命伤害，在恰烂分时说不定能帮你提升名次

推推棒：理想情况是我把对面美杜莎/光法/龙骑推过来秒掉，实际情况是把三星伐木机推到后排放两个D然后自爆喷把两星炸弹人推过来放全家桶喷
见仁见智，反正我打死都不会选了哭笑

很多时候BKB要比刷新好用，特别当对面也是大招流的时候，潮汐或者美杜莎能稳定放出大比什么都重要。

暂时就想到这些，后面有别的再补充。

以上是我的个人理解，不一定就对，欢迎大家交流。最后祝各位早日上分！

毫无保留，终极上分攻略

大家好我是果果，在这里分享一下我一路上分的思路。

首先要说的是在我玩了这么多局之后，我觉得刀塔霸业的吃鸡条件相较其它自走棋就目前的模式来说更看运气。为什么这么说呢？

首先我想讲一下，目前在我看来刀塔霸业的吃鸡条件:

1：天赋的选择
2：前期血量的控制
3：后期装备的选择。

这三点在我看来是对吃鸡影响最重要的。

天赋需要高级别罪犯。

血量在20波时保持在80点以上。

在后期装备中选到黑皇，刷新，烟斗等关键装备。

如果你都符合，恭喜你这把就吃鸡了。如果没有那你也稳定进入神仙打架的决赛圈。

终于上尖塔了，给大家分享一下攻略吧

总的来说，这个版本吃鸡的话，就是万物转大招，除非有2个相同的BUFF，比如双恶魔，双刺客细沙等等。
然后，主要说一下前中后期思路吧，我觉得具体打法应该看发牌员2333333，每一局都不一样。

前期：主要在于尽量胡2星,凑羁绊。德鲁伊，战士，刺客就很厉害。小鹿+大树，或者2星的小小，2星血魔都很厉害。总之，就是尽量凑对子就是了。争取连胜。

我觉得前期能够连胜很关键的点还有在于D牌：我觉得前期可以有策略的D一下的。比如，对子多了，比如不卡当前回合和下一回合的利息的情况下。举个例子你有26块钱，这回合不卡利息，下回合会有至少7块收入，那么就有33块钱了，可以D一下，提升战斗力。保证连胜。搜不到，也不卡两回合的利息。总之，就是算一下，规划一下，接下来几回合有多少钱，该怎么打。

然后关于上人口，连胜或者不卡利息，有东西可以上的话，就上人口，比如9回合上人口。不然就存钱，因为10回合打野怪，多吃利息。

看到冰女无脑留，卡利息宁愿卖其他的。

物品推荐：失去天恩，跳刀，有钱人烦恼，锁子甲，野兽buff，战士buff。走私犯我不是很喜欢，因为，会让失去天恩的回合数变少。

中期：中期在于配合选的天赋，构造比较厉害的阵容，可以是刺客，猎人，法师，骑士，野兽，都可以，（我不爱地精 = = =）。发牌员给什么就玩什么。但是，如果没有2个相同的阵容BUFF，后期都要转型的，所以大部分棋子都可以不升3。被血入，或者连胜，或者对子多的时候，可以D以下，感觉差不多了就上人口。

看到光法留一留。
物品推荐：高级别罪犯，跳刀，失去天恩。

后期：三种打法，有双BUFF，双恶魔，双刺客细沙，双野兽，可以继续走原来的套路。其他的就要转型。
有天恩就6天恩（可以是屠夫，狼人，COCO，光法，冰女，巫妖）+炸弹（带跳刀）+橙卡+双娜迦（法师少可以不用）来说，这个版本吃鸡的话，就是万物转大招，除非有2个相同的BUFF，比如双恶魔，双刺客细沙等等。

大招流：橙卡+控制+光法，冰女，巫妖。
我觉得在前期掉血少的情况下，后期尽量贪一点，能上10就尽量早上10。不然很难打过人家有失去天恩天赋和高级别罪犯的。比如，你有80血，发现2楼有高级别罪犯，但是没你肥。你可以算一下，多少回合能到10人口，该掉血就掉血，每回合掉15也可以接受。然后10人口搜橙卡，提升战斗力。

然后说一下，后期带小鱼人和美杜莎吧。美杜莎应该是更好的。但是，我比较喜欢小鱼人，因为没人抢更容易升星。小鱼人可以骗飞机大招。美杜莎有时候会给对面前排送能量。
物品推荐：高级别罪犯，扩大阵容，BKB，刷新。

不要拿辉耀and西瓦，给对面前排充能。

天赋和装备只能靠洗洗脸洗洗手去去非酋气息来解决了。

主要拿牌思路看到小鹿大树必拿，战士对子必拿。

前期如果能凑到2星小鹿和一个一星大树。

2个二星德鲁伊配上3个一星战士也很有战斗力。

又肉又有恢复，尤其是在前期普遍是小脆皮的情况下，能保持尽量少的掉血。

如果3战士中能有2星小小或者海牙。

或者前10波天胡来了狼人，海军，末日。那你这局基本能安稳度过前中期。

如果你更幸运到能抓到2星小鹿和2星大树。那基本可以安稳度过到20波。

三星大树或者三星小鹿，在中期的压制力。谁用谁知道！
