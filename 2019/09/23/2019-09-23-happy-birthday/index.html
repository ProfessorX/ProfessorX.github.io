蛤皮 波斯嘚

发表于 2019-09-23 分类于 Report

正儿八经的28岁了，老大不小了，该懂事了。是不是可以考虑把游戏戒了，做一点正儿八经的事情？——题记

反思

  • 钱赚的不够多。
  • 实力撑不起野心。
  • 大约的确不会回中东读书了吧，要读书可能读一个在职的研究生。

毕业后这些年

  • 阿联酋研究生没有读完，算是人生重大失误。
  • 非洲没有购置资产？
  • 中间一些职业规划出现了偏差，导致简历不是特别好看。

舒适区

  • 呜呜呜，哭唧唧没有卵用。
  • 至少要有一个5-10年的宏观规划，然后在战术方面执行。
  • 百年前所未有之大变局。

远虑近忧

  • 本职工作。做好。
  • 程序员养蛊。
  • 提早做打算。技术路线和管理路线，技术+CPA两条腿走路。
  • 不应盲目拼比，做好自己该做的事情。制定年度和季度计划，按照计划行事，并尽量保证每周、至少双周做一个reflections.
  • 搞好老同学之间的关系，有钱大家一起赚。

打铁还需自身硬。

向西

  • 向西边更西边前进。
  • 既要懂技术，又要懂业务。甚至要更加懂业务。站在客户和客户的客户角度思考问题。
  • 技术+CPA=战略忽悠。

创刊号-在信息洪流里找一个安静的漩涡

  • 九月 03, 2019
    大凡新刊物出现，总是需要一个创刊号开宗明义、申明立场、表达期许。我想这个习惯到了网络时代也难以免俗罢。

我们同时是幸运而不幸的，信息密度极大爆炸带来了前所未有的海量信息和便捷的获取渠道。只要愿意我们随时可以获得远超前人的信息，但恰恰因为如此，泛滥的信息背后是远低于前人的信噪比与思考深度。

更不用提人工智能，尤其是推荐算法这个可以让我们永远呆在舒适区的撒旦般的存在。拿起手机，打开抖音，你完全可以在上划下划和嬉笑之间度过数个小时而不自知。在反复灌输的、相似风格的视频、音乐、文章、up主的规训之下，思想被越来越限制在狭小的领地，无处腾挪，直至不再活跃。

而这显然不是我所希望的，如何利用好信息时代的优势，而规避信息时代的危害，让我们相比前人不仅仅由于科技整体的发展而优越，而是确乎在行为、思想、意识上更坚实地踏出向前的步伐，应当是自觉的行为。恰如第五位面壁者，章北海的父亲所说，要多想。

于是这便是这个博客的起源了，要多想，要多记录。关于生活的，商业的，社会的，我希望在信息洪流里找到一个安静的漩涡，如果能在这里发现那么一两颗珍珠，那便更好了。
