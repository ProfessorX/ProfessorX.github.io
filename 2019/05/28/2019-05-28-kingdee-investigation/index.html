金蝶ERP与智慧记研究

发表于 2019-05-28 更新于 2021-09-10 分类于 Technology

改革开放40年，一大批优秀中国企业快速发展壮大，并走向世界。”做强做优
做大”成为中国大企业在新时代肩负的新使命。人工智能、区块链、云计算、大数
据、物联网……这些不断涌现的新技术和当下不断变化的国际态势，给当下的中
国企业带来前所未有的挑战。在数字化浪潮下，企业如何利用数字化技术重构IT架
构、重构管理模式？这是新时代企业面临的选择，也是中国企业管理软件行业所
面临的选择。机遇与挑战并存，是发展也是担当，中国企业云服务市场必将迎来
一个新时代。

以下内容优先讨论服务于中小企业的免费软件金蝶智慧记以及其对业务功能的抽象建模。

金蝶云苍穹

数字化浪潮下，云计算已经成为这个时代的必然选择，金蝶云苍穹作为中国首
款自主可控和基于云原生架构的企业级云服务平台应运而生。金蝶云苍穹以“人
人”、“生态”、“体验”为设计理念，提供完整的企业级SaaS服务和PaaS服
务。金蝶云苍穹SaaS服务包括员工服务云、财务云、人力云、采购云、制造云、销
售云、物流云、渠道云等。金蝶云苍穹PaaS采用云原生技术，支持微服务、分布
式计算与存储、容器服务、多租户，并独创了云端动态领域模型，支持云端个性
定制，是领先的企业云平台。

在大众创业、万众创新的时代背景下，个体崛起、创业精英不断涌
现，企业应该更加尊重个体，分享成功。企业的信息系统也应该以人为核
心，提供人人财务、人人费用、人人绩效、人人营销等一系列围绕“人
人”的应用。金蝶云苍穹处处贯穿体现“我为人人，人人为我”的理念，
让每个个体能够实时、准确、透明掌握自己价值创造的全过程，从而激活
每一个个体，为企业和个人创造无限可能。

在互联网+的大时代背景下，人与组织都存在于一个生态体系下，实
现多元共生、协同发展、跨界融合。金蝶云苍穹旨在帮助企业在一个社会
化商业网络中创造价值，构建生态共同体；提供给客户生态化的产品解决
方案，建立连接伙伴的生态化平台，为企业提供一个全方面的生态化信息
系统。

以极致体验致敬每一个勇敢拥抱数字化转型的用户，打造媲美消费级
的生产力软件体验，是金蝶云产品体验设计的伟大愿景。为实现这一目
标，金蝶云苍穹在产品设计过程中，紧紧围绕目标用户在具体场景下的关
键任务目标，并将前沿科技与最新设计趋势进行融合（如：卡片化设计、
扁平化设计、 z 轴设计、情感化设计、响应性设计等），力求为用户创造
前所未有的极致体验。

随着会计工作的标准化和进一步细分，重复性的、规则性的会计事务
工作将会被财务机器人所取代，会计人员仅处理例外工作。同时随着IoT、
大数据等新兴技术的发展，未来的实物资产管理、财务数据的收集与核对
也将会由机器人自动处理。财务机器人将大幅提升会计核算效率，将会计
人员从繁琐的事务性工作中解放出来。

在无人会计的基础上，业务数据能及时准确地被转化为财务数据，每
个个体都是财务数据的提供者，即“人人皆财务”；同时，财务数据用于
赋能业务人员，每个个体能够实时、准确、便捷地获得个人价值相关数
据，实现个人价值的自我管理和提升，即“财务为人人”。

采购服务云

采购云以开放、智能、便捷为产品设计思想，旨在帮助企业寻源到最优合作资
源，实现高效协同，打造企业与供应商的共赢云平台。采购云通过连接多种采购
电商平台，支持商城采购、比价采购、战略采购多种模式，帮助企业降低采购成
本，并与供应商建立战略合作伙伴关系。

渠道云服务

渠道服务云为企业搭建了一个营销服务平台及高效协同平台，帮助经销商向服
务转型。渠道服务云包括面向企业端的B2B电商中心、销售助手，面向渠道端的
经销商门户、微分销。

金蝶云苍穹PaaS服务

金蝶云苍穹作为中国首款自主可控和基于云原生架构的企业级云服务平台，采
用了全新的分布式架构，由微服务架构、容器服务、DevOps等为代表的云原生基
础架构组成。

金蝶云苍穹微服务架构，使平台能够以更小颗粒度、更敏捷的方式提供服务，
不同的应用可以独立部署与互相隔离，服务之间通过轻量级API进行通信，在技术
上充分体现了”我为人人、人人为我”的设计理念；金蝶云苍穹容器架构，使平台真
正实现基于云资源的弹性伸缩，实现秒级自动启动与切换。金蝶云苍穹
DevOps，可以快速实现开发与运维一体化，加速软件系统的交付周期和交付质
量，可以帮助企业快速，持续，可靠，规模化地交付云应用。

金蝶云苍穹，提供了云端可视化低代码的开发平台，支持在多租户模式下也能
够为不同客户定制个性化应用。金蝶云苍穹融合了金蝶25年来积累锤炼的动态领
域模型，将这项金蝶自主知识产权的技术进行全新云化，完美实现了企业级标准
应用和个性化应用在云端的在线开发、调试、运行一体化，零代码、零安装、零
编译、零部署。金蝶云苍穹为不同客户、不同行业、不同伙伴、甚至个人开发者
等，提供企业级云应用的全生命周期的开放生态服务，构筑一个共商、共建、共
享的企业命运共同体。

开发服务云

基于金蝶独创的动态领域模型，金蝶云苍穹实现了数据模型、报表模型、表单
模型及流程模型等四大类模型的动态建模工作。金蝶云苍穹为云应用（SaaS服
务）的开发、部署、运行及运营提供支撑服务及管理的工具，开发平台包括了微
服务组件、开发服务、运行服务、服务管理、API服务框架、应用建模、云支撑服
务与运维服务等。
