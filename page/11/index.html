单病种付费

发表于 2019-12-30 更新于 2021-09-10 分类于 Report

首先是单病种付费的概念：单病种付费_百度百科

划重点：医疗机构的收入仅与每个病例及其诊断有关，而与医疗机构治疗该病例所花费的实际成本无关。

郭嘉任务，2020年105个单病种收费全面铺开。

105个病种涵盖了市级医院收入的半壁江山。

2019年1月，某地多家大医院汇总了过去一年的经营数据，去主管部门开会。

各家医院表示，如果严格执行105个单病种收费标准，各家医院普遍将损失6%以上的毛收入，全面面临入不敷出和大额亏损。有的手术只要开展，医院每做一台就要赔5万元左右。

会议持续了一天。至晚八点，风暴中心的会议主持人又饿又累，情绪激动的表示：“同志们，你们这样诉苦，我们的会议开到春节都开不完！就这样跟你们讲吧，105个单病种是郭嘉任务，你们只有适者生存的义务，没有任何讨价还价的余地！散会！”

2020 年你对你所在的行业和当下的社会生活有哪些建议？

发表于 2019-12-11 更新于 2021-09-10 分类于 Report

当前经济面临需求不足和通货紧缩的压力很大，拉动内需是“六稳”的关键。群众需要闲暇消费精神商品，需要房子安置物质商品。必须转变观念，加班常态化导致群众无暇休闲，造成内需萎靡，对经济百害无一利，需要严格限制。同时要保证劳动者的公休假和年假。有条件的地方可以考虑郊区化，利用新农村建设整合出的宅基地，让一部分城市居民在农村住小洋楼；让一部分居住条件优越的新农村居民凭借交通条件改善能够每日通勤进城工作。郊区化既能拉动内需，又能同时提高城乡群众的生活水平。

2035年建成社会主义现代化强国，届时群众的住房、教育、医疗等基本需求的负担如果还是比较重，社会主义现代化强国就无从谈起。

请考虑城市居民基本住房。提高富余房产的持有成本，推动不同产权、不同性质的富余房产根据市场规律和政府引导，纳入公租房体系；全面落实“租售同权”，让新市民和低收入群体能够得到基本住房保障。

请考虑产假与入托衔接，让公立基础教育下伸到幼儿八个月至一周岁。请考虑控制家庭的课外辅导负担，让父母可以在下班后去学校接孩子。请解决双职工家庭的后顾之忧。

请继续深化当前医改举措，进一步压低医疗物资的渠道成本，以投入推动机制建设，尽快形成完善的分级诊疗体系和按病种付费模式。
